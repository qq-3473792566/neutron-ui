# neutron-ui

## 特性

- **Vue 3 原生支持**：基于 Vue 3.2+ 和 Composition API 开发
- **TypeScript 支持**：完整的类型定义
- **按需引入**：支持按需加载，减少打包体积
- **主题定制**：支持 CSS 变量主题定制

## 项目结构

```
neutron-ui/
├── docs/                 # 文档
│   └── components/       # 组件文档
├── packages/             # 组件源码
│   ├── button/           # 按钮组件
│   ├── input/            # 输入框组件
│   ├── ...
│   └── index.js          # 入口文件
├── dist/                 # 构建输出
└── package.json
```
